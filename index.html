Solid Edge Discord Rich Presence


Download

How to install

  • Unzip "SolidEdge Discord RPC Installer.rar"
  • Execute "InstallDiscordRPC.ps1"
  • The "Release" folder will be removed from the directory - the installion is done.
  • Done.

How to enable/disable it

  • Note that the AddIn will already be enabled when installing it!
  • Open Solid Edge and head to the settings
  • Open the "Add-Ins" Subsetting
  • Enable/Disable it by checking/unchecking "SolidEdge.Community.RichPresenceSolidEdge"
